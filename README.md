# Runner
3명의 트레저 헌터를 조작하여 해적의 보물을 빼앗고 드넓은 바다와 해적선을 탈출하는 러닝 액션게임입니다.

## 시스템 소개
내일배움캠프 32조에서 Unity와 C# 스크립트를 사용하여 구현하였습니다.
기본적인 플랫포머 액션 기능과 아이템 기믹으로 진행됩니다.


## 개발 기간
+ 2025.02.21(금) ~ 2025.02.28(금)
+ 아이디어 작성
+ 코딩 테스트
+ 기능 구현
+ 발표


## 개발 환경
+ Unity: 2022.3.17f1
+ Visual Studio 2022
+ 아이디어 회의: Slack, Notion, Figma
